void UART_Putchar ( const uint8_t  data,
FILE *  stream 
)

Transmit a single character.

Note
For formatted IO streams implementation.
Blocks until UART is ready for transmission. To avoid blocking use UART_WriteReady(void) to check if UART is ready for transmission.
When sending newline, line feed is automatically appended.
Parameters
[in]data- the character that will be transmitted
stream- pointer to the FILE stream
See also
UART_WriteReady(void)
