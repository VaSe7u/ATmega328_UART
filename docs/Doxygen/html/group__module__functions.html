◆ UART_ReadReady()

UART_Status UART_ReadReady ( void  )

Check if there is new data available for reading.

Also returns the error status of the reception.

Returns
bitfield of the read ready status and the error status
Position Name Explanation
0 UART_STATUS_TRUE Successful reception
2 UART_STATUS_PARITY_ERROR Parity error
3 UART_STATUS_OVERRUN_ERROR Overrun error
4 UART_STATUS_FRAME_ERROR Frame error
Return values
0- no new data
1- successful reception
>1 - failed reception
See also
ReturnStatus

void UART_Putchar ( const uint8_t  data,
FILE *  stream 
)

Transmit a single character.

Note
For formatted IO streams implementation.
Blocks until UART is ready for transmission. To avoid blocking use UART_WriteReady(void) to check if UART is ready for transmission.
When sending newline, line feed is automatically appended.
Parameters
[in]data- the character that will be transmitted
stream- pointer to the FILE stream

UART_WriteReady(void)
